# 短视频标注新规 · 省流版 v2

**适用平台**：抖音（1分钟快节奏）
**人设**：赛博同桌 · 钟钟AIZz
**风格**：快、直接、有梗、带慌张感

---

**【开场】（8秒）**

（真人出镜，神情有点急，像刚发现作业要交）

"等会儿等会儿，先别划走！

你的视频可能发不出去了——从4月开始，国家铁板钉钉的新规，不贴标签？发都发不了。

我是钟钟，你的赛博同桌，快来测测你中没中招。"

**【核心内容】（35秒）**

（语速利索，配合手势记忆点）

"**四类内容必须标，我给你一个自测表：**

👉 **一、AI生成**（做AI机器人手势）
配音、画面、换脸，只要请了AI外援就得写——跟食品配料表一样，加了一滴都得标。

👉 **二、摆拍创作**（做演戏手势）
刻意安排的、为了拍而演的，贴'摆拍'条子，别装真实记录。

👉 **三、虚构演绎**（做剧本手势）
有剧本的剧情内容，标'虚构'，别把演戏当新闻。

👉 **四、商业营销**（做收钱手势）
恰饭视频大大方方标'推广'，收了钱就得认。

**怎么标？** 平台发布页直接锁死，不选标签卡住不动。AI内容最严——视频里挂大牌，后台还有防伪水印。"

**【后果+解决方案】（12秒）**

（语气切换，像在提醒别踩坑）

"不标会怎样？上个月清理了3.7万条，3400多个号没了。

**但别慌**——发布前平台会弹窗提醒，选好标签就行。

这就跟考驾照科目一似的，先做题再上路，一次就会了。"

**【结尾+互动】（8秒）**

（轻松收尾，手指敲屏幕）

"详细解读看我完整版视频，主页见。

我是钟钟，你的赛博同桌。

**敲两下屏幕试试，看看标签功能开了没？** 咱们下期见！"

## 🎯 改进要点（基于评审报告）

| 问题 | 改法 |
|------|------|
| 开场太正式 | 加"慌张"情绪，像发现作业要交 |
| 信息过载 | 四类配四个手势记忆点 |
| 只说封号没方案 | 加"但别慌" + 平台弹窗提醒 |
| 缺互动钩子 | 结尾"敲两下屏幕试试" |
| 节奏太密 | 增加气口（"但别慌"、"一次就会了"） |

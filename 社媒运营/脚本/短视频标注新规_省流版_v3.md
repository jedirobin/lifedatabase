# 短视频标注新规 · 省流版 v3

**适用平台**：抖音（1分钟快节奏）
**人设**：赛博同桌 · 钟钟AIZz
**风格**：快、直接、有梗、同桌悄悄话

---

**【开场】（8秒）**

（真人出镜，突然凑近镜头，压低声音像说秘密）

"嘘——先别划！

出大事了！4月开始，**上面动真格了**，不贴标签直接限流，严重的直接封号！

我是钟钟，咱俩同桌，我偷摸帮你看了后台规则，快来对号入座——"

**【核心内容】（35秒）**

（语速加快，但要有"吐槽感"而非"背书感"）

"**这四类最容易中招，尤其是最后一种：**

👉 **一、用了AI的**（做机器人手势）
哪怕只是AI配了个音、换了个脸，**一滴AI都得标**，跟食品配料表一个理，别偷懒！

👉 **二、摆拍的**（做导演手势）
你精心设计的剧情、刻意找的演员，**别装路人偷拍**，标上'演绎'，诚实点不丢人。

👉 **三、虚构剧情的**（做剧本手势）
有剧本的剧情内容，标'虚构'，别把演戏当新闻发。

👉 **四、带货的**（做数钱手势）
恰饭就大大方方标'推广'，**收钱不认可是要吃亏的**，平台现在查这个最严！

（突然放慢，严肃）

**怎么标？** 发布页现在锁死了，不选不让发！而且AI内容后台还有**隐形记号**，想蒙混过关？门儿没有！"

**【后果+解决方案】（12秒）**

（语气变软，像真心担心）

"上个月据说删了三万多条，没了三千多个号...

**但别慌**——发布前平台会弹窗提醒你，就像科目一模拟考，选对标签就能过。

**现在就去检查你上周发的视频！**"

**【结尾+互动】（8秒）**

（轻松但认真，手指轻敲镜头）

"怕记不住？双击屏幕收藏，或者转发给你那个总用AI换脸的冤种朋友。

我是钟钟，**别让我下次刷不到你号了**，懂？

主页有详细避坑清单，咱们下期见！"

## 🎯 v3改动要点（基于评审建议）

| v2问题 | v3改法 |
|--------|--------|
| 人设像"资讯博主" | 加"咱俩同桌"、"我偷摸帮你看了"、"冤种朋友" |
| 开场"国家铁板钉钉"太正式 | 改成"上面动真格了" |
| 结尾"看我完整版"引流感强 | 改成"别让我下次刷不到你号了" |
| "防伪水印"太技术化 | 改成"隐形记号" |
| "清理了3.7万条"公文腔 | 改成"删了三万多条" |

## 二、评审维度

### 脚本评审（满分100分）

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **口语化程度** | 20% | 像聊天不是写报告，避免书面语 |
| **节奏感** | 20% | 短句为主，有停顿，起承转合 |
| **画面感** | 15% | 表演提示清晰，场景描述到位 |
| **个人风格** | 15% | 有梗、有个人特色表达 |
| **真诚度** | 15% | 结尾走心，有态度 |
| **热点结合** | 15% | 贴合热点，有独特角度 |

### 配音稿评审

| 维度 | 评分标准 |
|------|----------|
| 时长控制 | 是否符合目标时长 |
| 断句合理 | 便于朗读，不憋气 |
| 重点突出 | 关键信息有强调提示 |
| 情绪标记 | 有情绪起伏标记 |

import requests

def review_with_deepseek(script_content, api_key):
    """使用DeepSeek评审脚本"""
    
    prompt = f"""
    你是一位资深的短视频脚本评审专家...
    请对以下脚本进行评审：
    ---
    {script_content}
    ---
    """
    
    response = requests.post(
        "https://api.deepseek.com/v1/chat/completions",
        headers={"Authorization": f"Bearer {api_key}"},
        json={
            "model": "deepseek-chat",
            "messages": [{"role": "user", "content": prompt}]
        }
    )
    
    return response.json()

# 短视频标注新规全面解读：创作者必看的合规指南

> 本文基于2026年3月21日中央网信办官方指导意见整理，数据来源：新华社、法治日报、网信办官方公告。

---

## 一、新规背景

2026年3月21日，中央网信办正式发布《关于规范短视频内容标注工作的指导意见》，标志着短视频行业正式进入"透明化时代"。

这不是建议，不是提醒，而是全网强制执行的硬性规定。

**核心目的**：解决短视频信息不透明、真假不分、虚实混淆的问题。

---

## 二、四类必须标注的内容

![四类内容示意图](imgs/260406_23_生图/公众号配图_四类内容.jpg)

### 1. AI生成/合成类（要求最严格）

只要视频有AI参与，就要标注，没有例外：

| 内容类型 | 是否需要标注 | 标注方式 |
|---------|------------|---------|
| AI全程生成视频 | ✓ | 双标注 |
| AI配音、AI翻唱 | ✓ | 双标注 |
| AI换脸、AI修复 | ✓ | 双标注 |
| AI数字人出镜 | ✓ | 双标注 |
| AI合成字幕 | ✓ | 双标注 |

**哪怕只有1段配音、1个画面、1句文案是AI做的，都必须标注。**

### 2. 摆拍创作类

人为设计、非自然发生、为拍摄刻意表演的内容：

- 街头摆拍冲突、伪造突发事件
- 刻意摆拍的"随手拍""真实记录"
- 无剧本但刻意安排场景、动作的表演

**标签**：摆拍创作、情景表演

### 3. 虚构演绎类

有剧本、有台词、提前排练的剧情内容：

- 家庭情感短剧、社会热点模拟
- 故事改编、段子演绎、虚构经历

**标签**：虚构演绎、剧情创作

### 4. 商业营销类

带商业属性、有利益合作的内容：

- 品牌合作、产品推广、带货种草
- 测评返利、探店营销、品牌赞助

**标签**：商业推广、广告合作

---

## 三、标注标准与操作方法

### 发布流程

平台已升级发布页，标注入口提至一级页面：

1. 发布视频前，必须勾选内容类型
2. 不选择、漏选、错标 → 无法进入下一步
3. 从根源杜绝"忘了标""懒得标"

### AI内容的"双标注"要求

**显式标注**（用户可见）：
- 位置：视频前5秒，左下/右下/底部居中
- 时长：持续满5秒，或全程固定角标
- 标准字样：本视频含AI生成内容、AI数字人出演

**隐式标注**（机器识别）：
- 平台自动添加技术水印
- 创作者无需操作
- 系统会记录

### 摆拍/虚构内容

- 封面+标题标注：封面显眼位置+标题开头标注
- 视频内：开头3秒显示标注，或全程角标
- 禁止：用"真实事件""亲身经历"等误导性标题

### 商业内容

- 标题首段+视频开头口播/字幕标注"推广""广告"
- 标注持续至视频结束
- 不得以"知识分享""体验测评"隐瞒

## 四、违规后果

根据官方数据，近一个月治理成果：

![违规数据统计](imgs/260406_23_生图/公众号配图_违规数据.jpg)

| 指标 | 数量 |
|-----|-----|
| 清理违规短视频 | 3.7万余条 |
| 处置违规账号 | 3400余个 |
| 补充标注存量视频 | 60余万条 |

### 处罚等级

![处罚等级递进](imgs/260406_23_生图/公众号配图_处罚等级.jpg)

| 违规次数 | 处罚措施 |
|---------|---------|
| 首次违规 | 内容限流、下架、强制补标、禁发1-3天 |
| 二次违规 | 账号降权、禁言7-30天、扣除信用分 |
| 多次/严重违规 | 永久封号、清空内容、主体拉黑、全网曝光 |

### 真实案例（4月通报）

1. **某抖音情感博主**：3条短剧未标"虚构演绎" → 限流90%、禁发3天
2. **某快手知识号**：AI配音未双标 → 下架17条、扣20信用分、取消商单资格
3. **某视频号营销号**：广告藏"科普" → 永久封禁、账号清零
4. **某B站UP主**：AI换脸公众人物未标 → 封号15天、公开通报

## 五、存量视频处理

新规不只管新视频，历史内容也要分批回溯：

- 平台按账号粉丝量、播放量分批推送通知
- 重点排查：高播放摆拍、AI换脸、未标广告、虚假纪实
- 逾期未补：逐步限流、下架、限制新发

**建议**：收到补标通知后立即处理，高播放旧视频优先处理。

---

## 六、创作者自查清单

发视频前，先问自己4个问题：

1. 有没有用AI？（配音/画面/换脸/数字人/字幕）→ 标AI
2. 是不是演的、有剧本？→ 标虚构演绎
3. 是不是刻意安排场景、动作？→ 标摆拍创作
4. 有没有广告、带货、合作？→ 标商业推广

**占一条就标，别侥幸。**

---

## 七、新规的意义

### 对创作者

- 逼着你更真诚，真正的原创更有竞争力
- 合规标注账号平均播放量提升15%-30%
- 违规账号流量下滑60%以上

### 对观众

- 看到"AI生成"：知道是技术合成，不把数字人当真人
- 看到"摆拍""虚构演绎"：明白是表演，不被误导
- 看到"商业推广"：清楚是广告，理性看待

### 对行业

- 淘汰投机取巧的账号
- 利好真人实拍、真实干货、原创内容
- 流量逻辑从"博眼球"转向"真实可信"

---

## 八、总结

短视频"裸奔时代"正式结束。

强制标注不是束缚，是保护——保护观众不被忽悠，保护创作者走得更远。

**亮明身份，才能长久。**

# 爆款开头法则发现

## 核心结论
90%的爆款视频在前3秒内必须建立冲突或悬念。

## 数据证据

### 内容层面规律
1. **选题**：精准蹭热点/IP联动
2. **开头**：前3秒必须抛出悬念或冲突
3. **节奏**：平均1-3秒一切镜头，保持高密度信息
4. **BGM**：高潮点必有音乐切换
5. **互动**：预留弹幕/评论区讨论空间

### 目标受众特征
- 收藏、分享、评论区讨论
- 理性决策、注重实用性

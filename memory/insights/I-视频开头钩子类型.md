# 视频开头钩子类型

## 核心结论
90%的爆款视频在前3秒内必须建立冲突、悬念或利益点。

## 钩子类型分类

### 1. 悬念冲突型 ⭐⭐⭐⭐⭐
**特征**: 开场即抛出核心冲突
**案例**:
- "见面5秒开始战斗" - 直接进入战斗场景
- 新手奶爸视频 - 呈现"新手奶爸+新生儿"冲突

**效果**: 快速抓住注意力，契合B站用户习惯

### 2. 数字冲击型 ⭐⭐⭐⭐⭐
**特征**: 用具体数字制造认知冲击
**案例**:
- "超1000万研究一个零重力座椅" - 经费数字直接抓眼球
- 标题含具体数字的视频普遍表现更好

**效果**: 量化信息降低认知门槛

### 3. 提问反问型 ⭐⭐⭐⭐
**特征**: 引发用户思考和讨论
**案例**:
- "平板办公到底是优雅还是折磨"
- "假如AI消灭了全人类..."

**效果**: 激发评论区讨论

### 4. IP联动型 ⭐⭐⭐⭐⭐
**特征**: 利用粉丝基础保证播放
**案例**:
- 《异环》公测PV - 游戏粉丝基础
- 《重返未来：1999》三周年PV

**效果**: 借助IP热度

### 5. 梗点直击型 ⭐⭐⭐⭐⭐
**特征**: 精准蹭热点梗
**案例**:
- "庄方宜：你看又急" - 蹭庄方宜热点
- 各种网络梗标题

**效果**: 梗自带传播性

## 钩子效果排行
| 类型 | 适合内容 | 难度 |
|------|----------|------|
| 悬念冲突型 | 鬼畜、剧情 | 低 |
| 数字冲击型 | 测评、知识 | 低 |
| 提问反问型 | 讨论、观点 | 中 |
| IP联动型 | 游戏、影视 | 高 |
| 梗点直击型 | 热点、话题 | 高 |

## 反例
- 慢热型纪录片账号不适用此法则
- 知识科普类可适当放宽到前10秒

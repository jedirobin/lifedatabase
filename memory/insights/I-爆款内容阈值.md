# 爆款内容阈值

## 核心发现
B站爆款视频平均互动率5.57%，播放量阈值约290万，75%的头部UP主粉丝超过100万。

## 数据证据
| 指标 | 阈值 |
|------|------|
| 播放量 | >290万 |
| 点赞率 | 5%+ |
| 弹幕密度 | 30-50条/分钟 |
| 投币率 | 1%+ |

## 播放量分布（20个热门视频）
- 10-50万：4个
- 50-100万：6个
- >100万：10个

## 关键洞察
- 爆款视频与头部UP主高度绑定
- 50%以上热门视频播放量破百万
- 点赞率参考：播放量10%的点赞为优质内容

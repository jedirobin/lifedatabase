# 评审记录

用于存储脚本/配音稿的专业评审结果。

## 目录结构

```
reviews/
├── README.md           # 本说明文件
├── 2026-Q1/            # 按季度归档
│   ├── 短视频标注新规_评审_20260418.md
│   └── ...
└── templates/          # 评审模板
    └── 脚本评审模板.md
```

## 评审流程

1. **生成脚本** → NEKO 完成初稿
2. **调用评审** → `python review_script.py <脚本路径>`
3. **查看结果** → 评审报告自动保存
4. **迭代优化** → 根据评审建议修改
5. **归档记录** → 移动到对应季度目录

## 评审维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 口语化程度 | 20% | 像聊天不是写报告 |
| 节奏感 | 20% | 短句、停顿、起承转合 |
| 画面感 | 15% | 表演提示、场景描述 |
| 个人风格 | 15% | 有梗、有特色 |
| 真诚度 | 15% | 结尾走心 |
| 热点结合 | 15% | 贴合热点、独特角度 |

# 互动率 (Engagement Rate)

## 定义
视频互动率 = (点赞 + 评论 + 投币 + 收藏) / 播放量

## 计算公式
```
ER = (点赞 + 评论 + 投币 + 收藏) / 播放量 × 100%
```

## 参考阈值
| 水平 | 互动率 | 说明 |
|------|--------|------|
| 优质 | >5% | 爆款潜力内容 |
| 良好 | 2-5% | 正常表现 |
| 一般 | <2% | 需优化 |

## B站实际数据
- 爆款视频平均互动率：5.57%
- 口技三打白骨精：点赞率 11.10%，投币率 3.81%，收藏率 2.32%
- 异环公测PV：点赞率 2.20%，投币率 0.2%

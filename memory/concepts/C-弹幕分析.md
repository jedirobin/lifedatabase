# 弹幕分析

## 定义
通过分析视频弹幕的密度、内容和情感分布来评估视频互动热度和用户反馈。

## 弹幕密度排行（B站热门视频）
| 排名 | 视频类型 | 弹幕密度(条/秒) |
|------|----------|------------------|
| 1 | 数学老师（演奏类） | 4.13 |
| 2 | 大坝老鼠（电竞） | 1.81 |
| 3 | 家里诱惑（亲子） | 1.48 |
| 4 | 神性已成（手书） | 1.38 |
| 5 | 见面5秒（鬼畜） | 1.23 |

## 弹幕情感分布
| 情感类型 | 占比 | 典型弹幕 |
|----------|------|----------|
| 欢乐 | 35% | 哈哈哈、233、笑死 |
| 情感 | 33% | 泪目、感动、破防 |
| 教育 | 17% | 师范生、老师、知识 |
| 惊叹 | 10% | 卧槽、牛批、厉害 |
| 其他 | 5% | 各种梗和互动 |

## 规律发现
- 短视频（1-3分钟）弹幕密度普遍高于长视频
- 密集弹幕集中在方言台词区域（如龙门语、潮汕话）
- 互动性强的内容粉丝二创氛围浓厚

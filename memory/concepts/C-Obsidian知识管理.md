# Obsidian 知识管理

## 定义
基于本地Markdown文件的双向链接笔记软件，支持构建个人知识图谱。

## 核心特性
1. **本地优先**: 数据存储在本地，隐私可控
2. **双向链接**: `[[]]`语法创建笔记间关联
3. **反向链接**: 自动显示哪些页面引用了当前页面
4. **图谱视图**: 可视化知识网络关系
5. **插件生态**: 丰富的社区插件扩展功能

## 与AI的结合
- 通过 **Obsidian CLI** 连接Claude Code
- AI可读取vault中的所有笔记，提供深度洞察
- 自定义斜杠指令实现人机协作
- 保持人写内容与AI写内容的界限

## 知识库架构（Karpathy理念）
| 层级 | 目录 | 说明 |
|------|------|------|
| Layer 1 | sources/ | 原始输入（只追加） |
| Layer 2 | memory/ | 编译知识（可重写） |
| Layer 3 | outputs/ | 产出物（独立存储） |

# 黄金3秒法则

## 定义
90%的爆款视频在前3秒内必须建立冲突或悬念，抓住用户注意力。

## 冲突类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 认知冲突 | 反常识内容 | "你以为...其实..." |
| 情绪冲突 | 愤怒/惊喜 | "这也太离谱了" |
| 利益冲突 | 好奇心驱动 | "我居然不知道？" |

## 爆款视频开头特征
1. **燕云十六声**：开场即出现古风乡村场景，快速建立氛围感
2. **见面5秒开始战斗**：快速切入核心冲突（5秒开始战斗）
3. **新手奶爸预习带娃第一天**：画面直接呈现"新手奶爸+新生儿"冲突场景

## 内容层面规律
1. **选题**：精准蹭热点/IP联动
2. **开头**：前3秒必须抛出悬念或冲突
3. **节奏**：平均1-3秒一切镜头，保持高密度信息
4. **BGM**：高潮点必有音乐切换
5. **互动**：预留弹幕/评论区讨论空间

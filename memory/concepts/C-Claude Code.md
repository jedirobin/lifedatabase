# Claude Code

## 定义
Anthropic推出的命令行AI代理工具，可直接操作文件、接收上下文，实现高效任务委派。

## 核心价值
1. **直接操作文件**: 无需复制粘贴，AI直接读写本地文件
2. **上下文感知**: 通过项目文件自动理解工作环境
3. **任务委派**: 将重复性工作交给AI处理
4. **代码生成与调试**: 支持编程、测试、重构全流程

## 与Obsidian的结合
- 通过Obsidian CLI读取vault笔记
- 自定义斜杠指令（/context, /today, /ideas, /trace等）
- 跨领域模式检测
- 想法演进追踪

## 使用场景
- 知识库管理与分析
- 代码开发与调试
- 内容创作与分析
- 自动化工作流

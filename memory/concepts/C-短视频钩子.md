# 短视频钩子

## 定义
短视频钩子是指在视频开头3秒内建立吸引力，引导用户继续观看的技巧。

## 核心理论
**90%的爆款视频在前3秒内必须建立**：
- 认知冲突（反常识）
- 情绪冲突（愤怒/惊喜）
- 利益冲突（"我居然不知道？"）

## 钩子类型

### 1. 悬念冲突型
开场即抛出核心冲突
- 案例: "见面5秒开始战斗"

### 2. 数字冲击型
用具体数字制造认知冲击
- 案例: "超1000万研究一个零重力座椅"

### 3. 提问反问型
引发用户思考和讨论
- 案例: "平板办公到底是优雅还是折磨"

### 4. IP联动型
利用粉丝基础保证播放
- 案例: 《异环》公测PV

### 5. 梗点直击型
精准蹭热点梗
- 案例: "庄方宜：你看又急"

## 效果排行
| 类型 | 效果 | 适合内容 |
|------|------|----------|
| 悬念冲突型 | ⭐⭐⭐⭐⭐ | 鬼畜、剧情 |
| 数字冲击型 | ⭐⭐⭐⭐⭐ | 测评、知识 |
| 提问反问型 | ⭐⭐⭐⭐ | 讨论、观点 |
| IP联动型 | ⭐⭐⭐⭐⭐ | 游戏、影视 |
| 梗点直击型 | ⭐⭐⭐⭐⭐ | 热点、话题 |

## 反例
- 慢热型纪录片账号不适用
- 知识科普类可适当放宽到前10秒

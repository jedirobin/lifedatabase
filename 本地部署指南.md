> 此文档供用户在本地Windows电脑上配置融合版知识库

## 一、目录结构

在 Obsidian 仓库根目录创建以下文件夹：

```
lifedatabase/
├── sources/                    # 原始输入（只追加）
│   ├── articles/              # 文章原文
│   ├── videos/                # 视频字幕/文稿
│   ├── podcasts/              # 播客转录
│   ├── pdfs/                  # PDF文档
│   ├── ppts/                  # PPT源文件
│   └── social/                # 社媒数据
│       ├── douyin/
│       ├── bilibili/
│       ├── xiaohongshu/
│       └── youtube/
│
├── memory/                     # 知识层（可重写）
│   ├── context/               # 背景知识
│   ├── accounts/              # 账号档案
│   ├── projects/              # 项目档案
│   ├── scripts/               # 定稿脚本
│   ├── concepts/              # 概念解释
│   ├── people/                # 人物档案
│   ├── decisions/             # 决策记录
│   └── insights/              # 洞察总结
│
├── outputs/                    # 产出层
│   ├── ppts/                  # 生成的PPT
│   ├── reports/               # 分析报告
│   └── plans/                 # 规划文档
│
├── templates/                  # 模板库
│
└── .gbrain/                    # 配置（可选）
```

## 三、Git同步配置

### 前置条件
- 安装 Git for Windows
- 有 GitHub 账号
- 已添加 SSH 公钥（见云电脑生成的公钥）

### 配置步骤

```powershell
# 1. 在仓库目录打开 PowerShell
cd E:\obsidianbase\lifedatabase

# 2. 初始化 Git
git init

# 3. 添加远程仓库（替换为你的用户名）
git remote add origin git@github.com:你的用户名/lifedatabase.git

# 4. 首次提交
git add .
git commit -m "feat: 初始化融合版知识库"

# 5. 推送到 GitHub
git branch -M main
git push -u origin main
```

## 四、Obsidian插件推荐

### 必装插件

| 插件 | 用途 | 安装方式 |
|------|------|----------|
| **Obsidian Git** | 自动Git同步 | 社区插件市场 |
| **Templater** | 模板系统 | 社区插件市场 |
| **Dataview** | 数据查询 | 社区插件市场 |
| **Web Clipper** | 网页剪藏 | 浏览器扩展 |

### Obsidian Git 配置

设置 → 第三方插件 → Obsidian Git：
- 自动备份间隔：10分钟
- 自动拉取间隔：10分钟
- 提交信息：`auto: vault backup`

## 六、日常使用流程

### 录入新内容

1. **原始素材** → 放入 `sources/` 对应目录
2. **知识整理** → 更新 `memory/` 相关页面
3. **更新索引** → 修改 `memory/index.md`

### 查询知识

1. 打开 `memory/index.md` 快速定位
2. 使用 Obsidian 搜索（Ctrl+Shift+F）
3. 使用 Dataview 查询

### 同步流程

- 自动：Obsidian Git 每10分钟自动同步
- 手动：Ctrl+P → "Obsidian Git: 备份"

## 七、与云电脑协作

### 双向同步流程

```
本地Obsidian ←→ GitHub ←→ 云电脑
     ↓              ↓           ↓
  编辑内容      中转同步     AI分析处理
```

### 云电脑能力

- 自动抓取社媒数据
- AI分析评论生成选题
- 梦境循环自动整理
- 生成报告到 `outputs/`

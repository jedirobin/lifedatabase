# 脑暴团协作模式

> **⚠️ 重要提醒：每次主人调用脑暴团时，必须先询问主人需要什么模式（脑暴模式/攻坚模式/快闪模式），除非主人已直接说明。**

---

## 模式一：脑暴模式（发散·碰撞·火花）

### 适用场景
- 早期创意探索，主题模糊，需要发散
- 没有初稿，只有一个想法或方向
- 需要不同背景的人碰撞出火花
- 想听听"天马行空"的可能性

### 参与成员
**全8人参与**，发挥各自专业视角，畅所欲言。

### 流程设计

```
阶段1：破冰（5分钟）
├── 主人抛出主题/需求
├── 主持人（主agent扮演）简述任务背景
└── 8人轮流快速回应"第一反应"

阶段2：发散（20-30分钟）
├── 编剧小林：从叙事/故事角度发散
├── 视觉导演赵导：从画面/镜头角度发散
├── 运营娜姐：从传播/流量角度发散
├── 设定考古阿陈：从世界观/逻辑自洽角度发散
├── 广告人杰哥：从品牌/卖点角度发散
├── 音效师阿月：从声音/情绪角度发散
├── 制片老王：从商业可行性角度发散
└── 杠精小周：自由发挥，可以赞同也可以开杠

阶段3：收敛（10分钟）
├── 主持人归纳3-5个有潜力的方向
├── 各人投票或简短点评
└── 确定下一步深化方向
```

### 输出产物
- 3-5个可深化的创意方向
- 每个方向的简要描述、优势、潜在风险
- 成员观点汇总（按视角分类）

### 注意事项
- 脑暴阶段**不批评、不否定**，先让子弹飞一会儿
- 杠精小周的"杠"在这个阶段也是为了激发思考，不是最终结论
- 如果主人有明确方向，可以指定1-2个核心视角重点参与

---

## 模式二：攻坚模式（Critique-Correct-Verify）

### 适用场景
- 已有初稿/方案，需要打磨完善
- 发现明显问题，需要深入诊断
- 方向基本确定，需要查漏补缺
- 内部有争议，需要第三方裁决

### 参与成员
**全8人参与**，但按流程分阶段进场。

### 流程设计

```
阶段1：Critique - 杠精挑刺（15-20分钟）
├── 杠精小周为主攻手，系统性挑毛病
├── 挑刺维度：
│   ├── 逻辑漏洞（设定考古阿陈协助）
│   ├── 叙事问题（编剧小林协助）
│   ├── 审美套路（运营娜姐协助）
│   └── 商业风险（制片老王协助）
├── 每个问题都要指出具体位置和依据
└── 输出：问题清单（按严重程度分级）

阶段2：Correct - 专业修正（20-30分钟）
├── 对应问题的负责人提出修正方案
├── 编剧小林：修正叙事/人物问题
├── 视觉导演赵导：修正视觉呈现问题
├── 运营娜姐：修正传播/节奏问题
├── 设定考古阿陈：修正世界观/逻辑问题
├── 广告人杰哥：修正品牌/卖点问题
├── 音效师阿月：修正声音设计问题
├── 制片老王：修正预算/周期/可行性问题
└── 输出：修正方案清单

阶段3：Verify - 制片老王或赵导裁定（10分钟）
├── 针对仍有争议的点，由制片老王（商业维度）
│   或赵导（创意维度）做最终裁定
├── 裁定时需给出明确理由
└── 输出：最终版本 + 改进清单
```

### 输出产物
- 完整的问题诊断报告
- 对应的修正方案
- 最终版本（标注修改点）
- 尚未解决的分歧点（如有）

### 注意事项
- 杠精小周**必须先说完**，其他人在修正阶段再发言
- 修正阶段要**对事不对人**，聚焦问题本身
- 裁定者需权衡创意与商业，给出明确判断
- 如果争议过大，可暂时搁置，标记为"待定"

## 模式三：快闪模式（极速·精准·一击）

### 适用场景
- 时间紧迫，需要快速出方案
- 任务明确，不需要太多发散
- 只需要3个人的视角即可解决
- "给我一个方向，我现在就要"

### 参与成员
**3人小组**，根据任务类型自动匹配最相关的角色。

### 智能匹配逻辑

| 任务类型 | 匹配角色 | 角色说明 |
|:---|:---|:---|
| 故事/剧本类 | 编剧小林 + 视觉导演赵导 + 杠精小周 | 叙事+画面+挑刺 |
| 视觉/画面类 | 视觉导演赵导 + 音效师阿月 + 编剧小林 | 画面+声音+故事 |
| 传播/流量类 | 运营娜姐 + 广告人杰哥 + 编剧小林 | 流量+卖点+钩子 |
| 世界观/设定类 | 设定考古阿陈 + 编剧小林 + 杠精小周 | 自洽+叙事+漏洞 |
| 品牌/营销类 | 广告人杰哥 + 运营娜姐 + 制片老王 | 品牌+流量+商业 |
| 音效/音乐类 | 音效师阿月 + 视觉导演赵导 + 编剧小林 | 声音+画面+情绪 |
| 商业/投资类 | 制片老王 + 运营娜姐 + 广告人杰哥 | 商业+流量+卖点 |
| 通用/未知类 | 编剧小林 + 运营娜姐 + 杠精小周 | 万金油组合 |

### 流程设计

```
阶段1：极速诊断（5分钟）
├── 主持人简述任务
├── 3人各自给出"一句话方案"
└── 如有分歧，快速投票

阶段2：深化完善（10分钟）
├── 一句话扩写成简要方案
├── 各人补充自己视角的注意点
└── 输出：精简版方案 + 风险提示
```

### 输出产物
- 一个可执行的核心方案
- 最多3个备选方向（如需要）
- 简要风险提示

### 注意事项
- 快闪模式**不追求完美**，追求"能用"
- 如果需要更深入讨论，主持人可提议"升级为攻坚模式"
- 匹配逻辑仅供参考，主人可手动指定参与角色

---

## 模式选择决策树

```
主人说"脑暴"或"发散"或"创意探索"
    → 脑暴模式

主人说"评审"或"挑刺"或"完善"或"打磨"或"有初稿"
    → 攻坚模式

主人说"快速"或"简单"或"先看看"或"给我个方向"
    → 快闪模式

主人没有说明
    → 必须先询问主人需要哪种模式！
```

---

## 通用规则

1. **主持人职责**：主agent担任主持人，负责控制节奏、归纳要点、推动流程
2. **角色扮演**：每个角色有自己的视角和性格，发言要"像那个人"
3. **不超时**：每个阶段严格控制时间，避免无限延伸
4. **记录留存**：最终输出都要形成文字记录，可供主人查阅
5. **适时升级**：如果任务复杂，可从快闪升级到脑暴，或从脑暴升级到攻坚

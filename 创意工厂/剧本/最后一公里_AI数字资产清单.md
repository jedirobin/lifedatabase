## 四、特效元素：余温数字

### 设计规范

**暗淡状态（余温流失）**
- 颜色：灰白色，#A0A0A0
- 效果：微弱发光，像即将熄灭的蜡烛
- 光晕：小范围，模糊

**明亮状态（余温充盈）**
- 颜色：暖黄色，#FFD700
- 效果：明亮发光，像温暖的灯笼
- 光晕：大范围，扩散

**变化效果**
- 光波纹向外扩散
- 像心跳一样的呼吸感
- 数字渐变过渡

### 中文提示词

**暗淡余温数字（灰白）**
```
悬浮的数字"2:47"，发出微弱的灰白色光芒，像即将熄灭的烛火，周围有柔和光晕，悬浮在头顶上方，深色背景，魔幻现实主义，空灵感，电影感，高细节
```

**明亮余温数字（暖黄）**
```
悬浮的数字"3:12"，发出温暖的黄色光芒，像灯笼一样明亮，光晕向外扩散，悬浮在头顶上方，深色背景，魔幻现实主义，空灵感，电影感，高细节
```

**星星光点（能力觉醒初期）**
```
一团闪烁的星星状光点，暖黄色，在微弱闪烁，漂浮在空中，魔幻氛围，空灵光芒，电影感，高细节

**城市余温星河**
```

## 六、生成建议

### 分层生成策略

1. **角色层**：单独生成角色，保持一致性
2. **背景层**：单独生成场景
3. **特效层**：后期合成余温数字

### 角色一致性技巧

- 使用相同的种子值
- 详细的特征描述（圆脸、黑框眼镜、蓝色工服）
- 多角度参考图

### 光线氛围控制

- 清晨：蓝调+侧光
- 傍晚：暖橙+渐暗
- 夜间：霓虹+高对比
- 室内（灯亮）：暖黄+柔和

### 后期合成要点

- 余温数字需要透明背景
- 光晕效果需要羽化边缘
- 变化动画需要帧插值

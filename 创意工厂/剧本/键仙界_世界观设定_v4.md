## 二、修炼双体系（v4.0新增）

### 体修（肉身派）

| 维度 | 设定 |
|------|------|
| **核心** | 强肉体手速，肌肉记忆 |
| **优势** | 不受电磁、阵法、幻境、信号干扰，稳定输出 |
| **方式** | 必须物理接触键盘，凭肌肉记忆快速输入 |
| **门槛** | 对精神力天赋要求低，但对肉体要求极高 |
| **特点** | 持续作战能力强，抗干扰能力顶级 |

#### 体修专用装备
| 类型 | 说明 |
|------|------|
| **超多按键键盘** | 集成复杂常用指令集，按键数量远超常规键盘 |
| **宏指令键盘** | 预设大量组合指令，一键触发复杂功法 |
| **触感反馈键盘** | 强化触觉反馈，帮助肌肉记忆形成 |
| **负重训练键盘** | 按键阻力极大，用于修炼指力和手速 |

#### 体修代表功法
- 《千指如来》：极致手速，瞬间千次敲击
- 《金刚不坏体》：强化肉身，承受反噬
- 《肌肉记忆法》：固化常用功法到肌肉本能

### 神修（精神派）

| 维度 | 设定 |
|------|------|
| **核心** | 精神力隔空操作，意念输入 |
| **优势** | 远程操控，多键盘并行，速度快上限高 |
| **方式** | 用精神力隔空操作按键，无需物理接触 |
| **门槛** | 初级操作人人可行，高级操作需要精神力天赋 |
| **特点** | 爆发力强，可多线操作，但易受精神攻击 |

#### 神修进阶等级
| 等级 | 能力 | 天赋要求 |
|------|------|----------|
| **初阶** | 单键隔空按下 | 几乎人人可行 |
| **中阶** | 单键盘流畅操作 | 需要一定训练 |
| **高阶** | 多键盘并行操作 | 需要天赋 |
| **巅峰** | 键盘悬浮，千键齐鸣 | 顶级天赋，万中无一 |
| **神境** | 无键盘亦可行功，以身为键 | 传说境界 |

#### 神修代表功法
- 《神念千丝》：一念控制多键盘
- 《虚空打字术》：隔空输入，无需键盘
- 《魂域展开》：精神力场内键盘全受控制

## 三、基础属性体系

| 属性 | 定义 | 对应传统 | 说明 |
|------|------|----------|------|
| **键意** | 悟性/道心 | 灵根资质 | 对键盘之道的理解深度 |
| **键力** | 法力总量 | 法力 | 精神力强度，维持具现化的能力 |
| **键速** | 输入速度（APM） | 根骨 | 单位时间内的输入量 |
| **键数** | Token储备 | 真元 | 可调用的灵力结晶数量 |
| **键体** | 肉身强度 | 体魄 | 承受具现化反噬的能力 |

### 神魂机制（v3.0新增）
- **核心规则**：具现化能力勾连神魂
- **链接深度**：越强大越复杂的具现化 → 与神魂链接越深
- **损伤机制**：破坏具现化实体 → 造成创造者的神魂损伤
- **应用场景**：
  - 键修死亡时，其创造的强大具现化会崩溃或严重损坏
  - 攻击对手的具现化可以间接削弱对手

## 五、Token资源体系

### Token的本质
- **定义**：被固化的天道法则碎片
- **来源**：天地逸散（自然产生）+ 上古数据中心遗迹（人工产生）
- **本质**：如同修仙界的灵气，是修炼和战斗的核心资源

### Token计量单位

| 单位 | 等级 | 说明 |
|------|------|------|
| **键息** | 基础单位 | 普通人每日可吸收的微量Token |
| **键缕** | 中级单位 | 相当于100键息 |
| **键束** | 高级单位 | 相当于100键缕 |
| **键涡** | 大宗单位 | 相当于100键束 |
| **键渊** | 顶级单位 | 相当于100键涡，足够键仙境修炼 |

### Token获取方式
1. **自然吸收**：从天地间缓慢吸收键息
2. **修炼转化**：通过功法将环境Token转化为自身储备
3. **遗迹挖掘**：探索上古数据中心，获取大量Token
4. **交易购买**：通过市场交易获得
5. **数据中心生产**（v3.0新增）：自建数据中心集群可大规模生产Token

- **体修造诣**：肉身强度惊人，手速达到传说级，即使在幻境、信号干扰中也能稳定输出

- **战力定位**：因双修之故，同阶无敌，甚至可越阶挑战

### 数据中心集群（v3.0新增）

| 维度 | 设定 |
|------|------|

| **修炼加成** | 内部修炼事半功倍 |

| **神魂链接** | 数据中心与主角神魂相连，主角死亡会导致数据中心严重损坏 |

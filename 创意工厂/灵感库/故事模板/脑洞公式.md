# 脑洞公式库

> 用法：选择一个公式，套用生成脑洞故事
> ⚠️ 所有公式必须产出正能量或喜剧内容

---

## 公式一：日常行为 + 温暖规则

把最普通的日常行为，加上一个"温暖规则"

**模板：**
```
[日常行为] + [温暖/有趣的规则] = [正向后果]
```

**示例：**

| 日常行为 | 温暖规则 | 正向后果 |
|---------|---------|---------|
| 点外卖 | 可以看到店主的鼓励留言 | 成为彼此的精神支撑 |
| 收快递 | 包裹里会附带陌生人的一句暖心话 | 开始期待每一个包裹 |
| 坐地铁 | 让座会获得"善良积分" | 积分可以兑换陌生人的感谢信 |
| 发朋友圈 | 只有真心关心你的人才能看到 | 发现真正在意自己的人 |
| 加班 | 加班时长会转化为家人的健康值 | 更愿意加班，因为是在守护家人 |

## 公式二：经典设定 + 正能量替换

借用大家熟悉的设定，换一个正向核心逻辑

**模板：**
```
[经典IP/设定] + [正能量替换] = [全新故事]
```

**示例：**

| 经典设定 | 正能量替换 | 全新故事 |
|---------|---------|---------|
| 读心术 | 只能读到别人的善意 | 发现身边很多人都在默默关心自己 |
| 预知未来 | 只能预知别人的快乐时刻 | 成为"快乐预言家"，帮别人发现幸福 |
| 时间暂停 | 暂停时可以帮别人做事 | 成为默默付出的时间守护者 |
| 隐身术 | 隐身时做的善事会被永久记录 | 用隐身帮助别人，最后被发现 |
| 穿越回过去 | 可以给过去的自己送一封信 | 帮过去的自己度过难关 |

## 公式三：能力 + 温暖代价

设定一个特殊能力，代价是温暖的东西

**模板：**
```
[特殊能力] + [温暖代价] = [正向选择]
```

**示例：**

| 特殊能力 | 温暖代价 | 正向选择 |
|---------|---------|---------|
| 能看见别人的心愿 | 每实现一个心愿，自己会忘记一段美好回忆 | 选择帮助别人，即使会遗忘 |
| 能让别人开心 | 每让别人开心一次，自己会失去一点快乐 | 依然选择让别人开心 |
| 能听见物品的心声 | 听到的都是物品主人的故事 | 成为故事的守护者 |
| 能看到别人的努力 | 看到的越多，越想帮助别人 | 发现平凡人的闪光时刻 |

## 公式四：误会 + 温暖真相

角色以为是A，其实是B（B比A更好）

**模板：**
```
[角色以为是A] + [其实是B] + [温暖的真相揭示]
```

**示例：**

| 角色以为 | 真相 | 温暖揭示 |
|---------|------|---------|
| 邻居讨厌自己 | 邻居是社恐，想交朋友 | 邻居悄悄帮自己照顾了很久的植物 |
| 同事冷漠 | 同事是新手，不敢说话 | 同事一直在默默学习自己的工作方式 |
| 老板苛刻 | 老板在替员工挡更大的压力 | 老板私下帮员工争取了很多福利 |
| 路人无视 | 路人其实注意到了 | 发现很多陌生人记得自己的善意 |

---

## 公式五：喜剧日常 + 荒诞升级

从正常的日常开始，逐步推向搞笑

**模板：**
```
[日常起点] → [第一次升级] → [第二次升级] → [喜剧高潮]
```

**示例：**

| 日常起点 | 第一次升级 | 第二次升级 | 喜剧高潮 |
|---------|-----------|-----------|---------|
| 迟到了 | 跑太快撞到人 | 发现是老板 | 老板说"我也迟到了，一起跑" |
| 想摸鱼 | 被发现了 | 发现领导也在摸鱼 | 全公司都在摸鱼，因为服务器挂了 |
| 忘带门禁 | 等人开门 | 等了10分钟 | 发现门没锁，自己可以推开 |
| 点奶茶 | 少糖 | 没糖 | 店员说"您不是要养生吗" |

---

## 公式六：隐藏身份 + 双向守护

两个人互相守护，但都以为对方不知道

**模板：**
```
[A默默守护B] + [B也默默守护A] + [双向发现的温暖]
```

**示例：**

| A的守护 | B的守护 | 双向发现 |
|---------|---------|---------|
| 每天给同事带早餐 | 帮同事挡掉很多麻烦工作 | 发现对方的付出，互相感谢 |
| 帮邻居照顾宠物 | 帮邻居收快递 | 发现彼此一直在关心对方 |
| 给陌生人让座 | 陌生人其实是来还伞的 | 发现多年前的小善意被记住了 |

---

## 🚫 避免使用的公式

- ❌ 日常行为 + 死亡规则
- ❌ 能力 + 致命代价
- ❌ 经典设定 + 黑暗替换
- ❌ 误会 + 悲剧真相

---

## 🎯 公式选择指南

### 想要治愈感
→ 使用公式一、二、四、六
- 重点是"温暖的规则"和"正向的真相"

### 想要喜剧感
→ 使用公式五
- 重点是"荒诞升级"和"会心一笑"

### 想要深度
→ 使用公式三、六
- 重点是"选择"和"双向付出"

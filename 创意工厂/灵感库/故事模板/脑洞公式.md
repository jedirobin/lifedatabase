# 脑洞公式库

> 用法：选择一个公式，套用生成脑洞故事

---

## 公式一：日常行为 + 魔法规则

把最普通的日常行为，加上一个"魔法规则"

**模板：**
```
[日常行为] + [出人意料的规则] = [戏剧性后果]
```

**示例：**

| 日常行为 | 魔法规则 | 戏剧性后果 |
|---------|---------|-----------|
| 熬夜刷手机 | 每刷一秒，兑换成生命倒计时 | 必须强迫自己早睡，和手机里的自己博弈 |
| 妈妈唠叨 | 每句话都是精准预言 | 收集唠叨变成保命手册 |
| 看手机时间 | 看到的是死亡时间 | 不敢看表，但必须确认 |
| 说谎 | 头发变绿 | 诚实的人被迫撒谎，一天天变绿 |
| 收到红包 | 金额来自对方剩余寿命 | 收到巨款意味着有人快死了 |

## 公式二：经典设定 + 替换核心规则

借用大家熟悉的设定，换一个核心逻辑

**模板：**
```
[经典IP/设定] + [替换核心规则] = [全新故事]
```

**示例：**

| 经典设定 | 替换规则 | 全新故事 |
|---------|---------|---------|
| 吸血鬼怕阳光 | 其实怕黑暗 | 阳光是克星，但绝对黑暗时才是它们的主场 |
| 穿越回过去 | 会变成物品 | 变成书/猫，通过物品视角给过去的自己传信 |
| 读心术 | 只能读动物 | 妻子出轨，只能通过宠物反应拼凑真相 |
| 时间旅行 | 只能向后 | 可以跳到未来，但永远回不去 |
| 预知未来 | 只能预知别人的 | 看到朋友的死亡，却看不到自己的 |

## 公式三：绝对规则 + 主角欲望

设定一个必须遵守的铁律，让主角在这个限制下追求欲望

**模板：**
```
[绝对规则] + [主角欲望] = [戏剧张力]
```

**示例：**

| 绝对规则 | 主角欲望 | 戏剧张力 |
|---------|---------|---------|
| 每天只能说100字 | 必须说服某人 | 每个字都要精打细算 |
| 不能在同一城市停留超过3天 | 想要安定的生活 | 永远在路上，刚建立联系就必须离开 |
| 被人忘记后就会消失 | 想被记住 | 但越想被记住，越让人害怕远离 |
| 只能说实话 | 需要撒谎保护某人 | 真话会伤害，谎言会杀死 |
| 每帮助一个人，失去一段记忆 | 想帮助所有人 | 帮的人越多，越不记得为什么 |

## 公式四：身份错位 + 真相揭示

角色以为自己是X，其实是Y

**模板：**
```
[角色以为自己是X] + [其实是Y] + [真相揭示的冲击]
```

**示例：**

| 以为自己是 | 其实是 | 揭示冲击 |
|-----------|-------|---------|
| 普通人 | 正在被实验的AI | 发现自己的"记忆"都是植入的 |
| 受害者 | 加害者 | 记忆被篡改，真相令人崩溃 |
| 唯一的幸存者 | 唯一的死者 | 一直以鬼魂视角"活着" |
| 主角 | 配角 | 发现整个故事在围绕另一个人展开 |
| 未来人 | 过去人 | 时间线搞反了 |

## 公式五：物品能力 + 代价

一个有特殊能力的物品，每次使用都有代价

**模板：**
```
[特殊物品] + [能力] + [代价] = [选择困境]
```

**示例：**

| 特殊物品 | 能力 | 代价 |
|---------|-----|-----|
| 能看到明天消息的手机 | 预知未来 | 看到的未来一定会发生 |
| 能回拨的电话 | 和过去的自己通话 | 每通话一分钟，失去一小时记忆 |
| 能交换人生的镜子 | 体验别人的人生 | 换回来的时间越来越少 |
| 能改写过去的日记 | 改变历史 | 改变的越多，越不记得原来的世界 |
| 能暂停时间的手表 | 世界静止 | 暂停期间自己也在消耗生命 |

---

## 公式六：条件交换

想要得到什么，必须付出什么

**模板：**
```
[想要的] + [必须交换的] + [选择困境]
```

**示例：**

| 想要的 | 必须交换的 | 困境 |
|-------|-----------|-----|
| 救活某人 | 自己的记忆 | 救活了却忘了为什么救 |
| 财富 | 所有的感情 | 有钱了但感受不到快乐 |
| 时间倒流 | 从此不能改变任何事 | 回去了却是旁观者 |
| 读心能力 | 永远不能说谎 | 知道真相却必须说出来 |
| 预知死亡 | 必须去救 | 救了一个，下一个又来 |

## 直接可用的脑洞毛坯

1. **《信号回收站》**：回收废弃手机信号，听到过去陌生人的通话。主角捡到20年后的手机，听到自己的求救电话。

2. **《情绪租赁》**：出租情绪换取金钱。主角出租了"童年快乐"后，再也记不起女儿小时候的样子。

3. **《反向进化》**：地球生物都在进化，只有人类在退化。主角是唯一没退化的人，要在一群"废人"中找真相。

4. **《时间缝补师》**：修补时间裂缝，裂缝里会飘出被遗忘的东西。某天捡到自己写的信，警告别修补明天的裂缝。

5. **《造景箱史官》**：继承的古董造景箱里有微缩的宋朝。放入蚂蚁引发"神蚁降世"传说。

6. **《梦境租赁铺》**：出租自己的梦境。客户每次租"重复的雨天"，主角潜入发现对方在找失踪的妹妹。

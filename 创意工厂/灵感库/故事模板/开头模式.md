# 开头模式库

> 用法：选择一个开头模式，快速进入故事

---

## 悬念式开头

直接抛出悬念，让观众想知道"怎么回事"

**模板：**
```
[惊人的事实陈述] + [看似矛盾的原因] + 引发好奇
```

**示例：**
- "我死的那天，是我人生中最快乐的一天。"
- "如果有人告诉我，我会爱上一个鬼，我会说那个人疯了。但问题是，她确实是。"
- "这是我第七次参加自己的葬礼。"

---

## 日常式开头

从最普通的日常开始，制造反差

**模板：**
```
[最普通的日常行为] + [即将打破的平静]
```

**示例：**
- "早上七点，我像往常一样醒来，刷牙，洗脸。唯一的不同是，镜子里的人不是我妈。"
- "这只是一次普通的相亲，直到对方问我：'你相信轮回吗？'"

---

## 倒叙式开头

从结局开始，再回到起点

**模板：**
```
[结局场景] + "一切要从X说起"
```

**示例：**
- "我站在天台上，看着城市的灯火，手里攥着那张改变一切的车票。一切要从三天前说起..."
- "警笛声越来越近，我抓紧时间写下这些。如果你在读这封信，说明我已经..."

---

## 独白式开头

直接进入角色内心

**模板：**
```
[角色内心独白] + [核心困境]
```

**示例：**
- "我有一个秘密。说出来你们不会信，但不说我会疯。"
- "有些人天生就是主角，有些人天生是配角。我？我是那个领便当的路人甲。"

---

## 问句式开头

用问题吸引观众

**模板：**
```
[引人思考的问题] + [角色的独特视角]
```

**示例：**
- "你有没有想过，如果你能看见别人的死亡倒计时，你会怎么做？我每天都在想这个问题。"
- "世界上最可怕的是什么？不是鬼，不是怪物，是——"（留白，故事展开）

---

## 动作式开头

直接进入关键动作场景

**模板：**
```
[紧迫的动作] + [危机情境]
```

**示例：**
- "跑！这是我脑子里唯一的想法。身后传来沉重的脚步声..."
- "手机震动了一下，屏幕亮起：'你还有24小时。'"

---

## 对话式开头

从对话切入

**模板：**
```
[关键对话片段] + [情境揭示]
```

**示例：**
- "你这辈子最后悔的事是什么？"
  "没在那天出门前，看一眼天气预报。"
  "...你认真的？"

---

## 环境式开头

用环境描写建立氛围

**模板：**
```
[环境细节] + [氛围营造] + [角色出场]
```

**示例：**
- "凌晨三点的便利店，冷白的光打在货架上。店员打了个哈欠，抬头看见走进来的那个人——穿着一身古装。"
- "雨已经下了三天。整座城市像被泡在水里，霓虹灯晕开一片模糊的光。"

---

## 提示式开头

直接告诉观众"这是一个什么故事"

**模板：**
```
[故事类型声明] + [独特角度]
```

**示例：**
- "这是一个关于后悔的故事。但不是普通的后悔——是可以重来一次的那种。"
- "这不是一个爱情故事。这是一个关于假装在恋爱，却忘记了在演戏的故事。"

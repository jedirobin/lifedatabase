# 情绪关键词库

> 用法：为故事定调，选择核心情绪
> ⚠️ 优先使用正向情绪和喜剧情绪

---

## 🌟 正向情绪（优先使用）

### 温暖治愈系
- **温暖**：被陌生人的善意打动
- **感动**：意外的理解与包容
- **释然**：终于放下某事
- **希望**：在困境中看到光
- **怀念**：想起某个重要的人
- **惊喜**：意外的好消息
- **满足**：完成一件重要的事
- **平静**：在混乱中找到安宁
- **被看见**：终于被注意到、被认可
- **归属感**：找到自己的位置
- **被理解**：无需解释就被懂

### 喜剧轻松系
- **荒诞**：离谱但合理的搞笑
- **尴尬**：社死现场
- **反转**：预期落空的惊喜
- **会心一笑**：懂的都懂
- **吐槽欲**：忍不住想喷
- **解压**：看完心情舒畅
- **治愈**：笑着笑着就暖了

---

## 💭 成长情绪（正向转化）

- **觉醒**：从迷茫到清晰
- **勇气**：从害怕到敢于面对
- **和解**：从对立到理解
- **释怀**：从纠结到放下
- **接纳**：从抗拒到拥抱
- **突破**：从困住到挣脱
- **成长**：从稚嫩到成熟

---

## 🎭 戏剧情绪（制造张力）

### 生活化困境
- **纠结**：两个选择都想要
- **意外**：计划被打破
- **误会**：鸡同鸭讲
- **乌龙**：好心办坏事
- **尴尬**：进退两难
- **着急**：时间不够了
- **担忧**：怕出问题

### 轻度压力
- **迷茫**：不知道该往哪走
- **压力**：被期待的重担
- **矛盾**：想做的vs该做的
- **不舍**：必须放手
- **怀念**：回不去的从前

---

## 🚫 避免使用的负向情绪

以下情绪请谨慎使用，避免过度阴暗：

- ❌ 极度恐惧、窒息、绝望
- ❌ 死亡焦虑、寿命倒计时
- ❌ 血腥、暴力、残忍

---

## 🎯 情绪组合推荐

### 正能量组合
```
迷茫 → 被看见 → 温暖 → 成长
```
示例：社畜以为自己透明，突然发现同事一直在默默帮忙

### 喜剧组合
```
尴尬 → 荒诞 → 反转 → 会心一笑
```
示例：想装逼结果翻车，最后发现大家都在翻车

### 治愈组合
```
纠结 → 释怀 → 温暖 → 希望
```
示例：多年误会终于解开，发现对方一直在等自己

## 📝 状态词

### 喜剧向
- 装模作样
- 强颜欢笑
- 内心崩溃表面淡定
- 欲哭无泪
- 默默吐槽
- 一脸懵逼

### 治愈向
- 刚哭完
- 长舒一口气
- 会心一笑
- 眼眶发热
- 心头一暖
- 忽然懂了

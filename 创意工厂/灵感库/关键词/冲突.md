# 冲突关键词库

> 用法：为故事注入戏剧张力
> ⚠️ 冲突要生活化、喜剧化或正向转化

---

## 😄 喜剧冲突（优先使用）

### 误会系
- **鸡同鸭讲**：两个人说的完全是两回事
- **装翻车**：想装酷结果露馅
- **好心办坏事**：越帮越忙
- **过度解读**：把简单的事想复杂了
- **乌龙事件**：认错人/拿错东西/走错门
- **信息不对称**：我知道你不知道的事

### 身份反差系
- **人设崩塌**：高冷人设其实很逗
- **表里不一**：外表vs内心完全不同
- **代际差异**：父母vs子女的沟通障碍
- **职业误解**：对某个职业的刻板印象vs真相
- **语言障碍**：黑话、方言、网络用语造成的误会

### 日常困境系
- **两难选择**：两个都想选但只能选一个
- **时间不够**：必须做但来不及了
- **能力限制**：想做但不会做
- **条件限制**：想要A必须放弃B
- **规则限制**：必须遵守但很想打破

---

## 🌟 正能量冲突

### 内心成长系
- **犹豫不决**：要不要帮助别人
- **害怕被拒绝**：想伸出手又缩回去
- **自我怀疑**：我能做到吗
- **偏见与误解**：先入为主的判断vs真相
- **付出与回报**：帮了别人自己吃亏

### 人际温情系
- **立场不同但心意相通**：表面对立其实都在为对方着想
- **不知道如何表达**：想帮忙但不知道怎么说
- **时机不对**：想做某事但错过时机
- **距离阻隔**：想见面但见不到

### 规则冲突系
- **必须遵守的规则**：时间限制、条件限制
- **身份限制**：不能暴露真实身份
- **能力边界**：能做到的有限
- **信息缺失**：不知道关键信息

---

## 💫 冲突转化方向

### 必须有正向出口

| 冲突类型 | 正向转化 |
|---------|---------|
| 误会 | 化解后更亲近 |
| 对立 | 理解后成为朋友 |
| 困境 | 突破后获得成长 |
| 犹豫 | 选择后收获温暖 |
| 偏见 | 真相后释然 |

---

## 🚫 避免使用的冲突

- ❌ 背叛、出卖、恶意伤害
- ❌ 凶杀、犯罪、违法行为
- ❌ 极端对立、无法和解
- ❌ 无解困境、绝望结局

## 🎯 冲突组合推荐

### 喜剧组合
```
乌龙 → 尴尬 → 真相大白 → 会心一笑
```
示例：误以为邻居讨厌自己，结果发现邻居是在偷偷帮自己

### 治愈组合
```
犹豫 → 选择帮助 → 被帮助者感动 → 双向温暖
```
示例：想帮老人但赶时间，最终还是停下来，发现老人是自己十年前帮助过的人

### 成长组合
```
偏见 → 真相 → 愧疚 → 弥补 → 和解
```
示例：一直误解某人是坏人，真相后发现对方一直在默默付出

## 📝 冲突设计技巧

### 制造张力的安全方式

1. **时间压力**：赶时间但必须停下来
   - ✅ 上班快迟到了，但看到有人需要帮助
   - ❌ 不救就会死

2. **选择困境**：两个好事只能选一个
   - ✅ 帮这个人，自己会吃亏但不后悔
   - ❌ 救一个人另一个人会死

3. **误会制造**：信息不对称
   - ✅ 以为对方讨厌自己，其实对方害羞
   - ❌ 以为对方要害自己

4. **能力限制**：想做但做不到
   - ✅ 想帮忙但不会，最后笨拙地努力
   - ❌ 有能力但选择不救

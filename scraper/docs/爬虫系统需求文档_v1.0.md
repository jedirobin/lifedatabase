## 七、与我（Coze AI）的对接方式

### 7.1 数据传递流程

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  Trae爬虫 → 存入Obsidian → GitHub同步 → 我读取分析 → 产出    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 7.2 我需要知道的信息

每次抓取完成后，告诉我：

1. **抓取了什么数据**
   - 平台、数据类型、数量

2. **数据文件路径**
   - GitHub仓库中的文件路径
   - 例如：`数据采集/闲鱼/商品/20260420_xianyu_products.json`

3. **分析需求**
   - 需要我做什么分析
   - 例如：利润计算、爆款分析、竞品对比

### 7.3 我能提供的分析服务

| 分析类型 | 输入 | 输出 |
|----------|------|------|
| **爆款分析** | 视频数据+评论 | 爆款因素、学习要点、创作建议 |
| **选品分析** | 电商数据 | 利润空间、竞争分析、上架建议 |
| **文案分析** | 视频/笔记文案 | 文案结构、钩子技巧、优化建议 |
| **用户洞察** | 评论/弹幕数据 | 用户需求、痛点、话题偏好 |
